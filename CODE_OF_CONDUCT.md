# Code of Conduct

## Our Pledge

We want Codex Chrome Bridge to be a respectful, practical, and safe project for people building local browser tooling.

Everyone participating in this project is expected to:

- Be respectful and constructive.
- Assume good intent, but accept feedback when impact differs from intent.
- Keep discussions focused on the project and its users.
- Avoid harassment, personal attacks, discriminatory language, and public sharing of private browser data.

## Scope

This code of conduct applies to all project spaces:

- GitHub issues, pull requests, discussions, and comments.
- Security reports and maintainer conversations.
- Any community spaces that explicitly reference this project.

## Enforcement

Maintainers may remove comments, close issues, block accounts, or restrict participation when behavior harms the project or its users.

If you need to report a conduct issue, contact the maintainer privately. Do not include private account data, cookies, tokens, or dashboard screenshots in public reports.

## Attribution

This code is inspired by common open-source community standards and adapted for a browser-privacy-sensitive project.
